# Notes Created
```dataview  
List FROM ""  

WHERE file.cday = date(today)  

SORT file.ctime asc
```


```dataview  
TABLE WITHOUT ID  

file.link AS "Note Name", file.mtime AS "Last Updated", length(file.inlinks or []) AS "Links"  

FROM ""  

WHERE dg-publish = true  

SORT file.mtime DESC  

LIMIT 10
```
